# 重要觀念

## 機密性(Confidentiality)
```
Ｃ： 機密性(Confidentiality)：資訊不得被未經授權之個人、實體或程序所取得或
揭露的性質。
```
## 完整性  (Integrity
```
資訊安全的三個基本要點之一，指在傳輸、儲存資訊或資料的過程中，確保資訊或資料不被未授權的篡改或在篡改後能夠被迅速發現。
Ｉ：完整性(Integrity)：對資訊之精確與完整安全保證的性質。
```
##　可用性   (Availability
```
Ａ：可用性(Availability)：已授權實體在需要時可存取與使用資訊之性質。
```
