# 怎麼執行滲透測試
## 滲透測試需要有
```
● 攻擊者機器
● 多台攻擊者機器
● 被攻擊的web
● networks 
● 等等
```
# 好處
```
● 在當前操作系統中安裝多個操作系統
● 每台機器都有自己的資源和功能，就像是一台真正的機器
● 測試更容易
● 更容易修復
● 與主機os隔離
```
# 最基本的linux語言
```linux
● ls         列出當前目錄下的所有文件
● man ls     例如:help查看指令
● ls -l      <-這個就是L 是列出當前目錄下的所有文件但比較仔細 
● ls --help  幫助
● cd 文件名   就會跑到文件裡面
● pwd        可以看到當前的目錄
● cd ..      回到上個目錄
● clear      清空
以下是Metasploitable 使用的
● ifconfig                查IP
● Metasploitable nano ... nano是編輯文件的程序
● sudo                    以管理員執行
```
