# 怎麼在目標網站找文件和目錄
```
1.先開 metasploitable 在裡面輸入 cd /var/www  我們知道常用的外部服務器都藏在var www文件夾下面 所以我們利用cd 到達這個目錄
2.過來到了 var/www下面後 再輸入 ls -la  -la這個指令可以把隱藏的文件給顯示出來
```

